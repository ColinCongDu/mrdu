# fs 模块

## `fs.readdirSync(path, options）`

用于同步读取指定目录的内容。它会返回一个包含目录中所有文件和子目录名称的数组。

### 参数说明

-   `path`（必需）：

要读取的目录路径。
可以是绝对路径或相对路径。

-   `options`（可选）：

用于指定返回值的格式。可以是一个字符串或对象，支持以下选项：
    -  `encoding`：
        指定文件名的字符编码，默认是 `'utf8'`。
    -   `withFileTypes：`
        如果设置为 `true`，返回的数组将包含 `fs.Dirent` 对象，而不是字符串。

### 返回值

- 默认情况下，返回一个包含目录中所有文件和子目录名称的字符串数组。
- 如果 `withFileTypes` 设置为 `true`，返回一个 `fs.Dirent` 对象的数组。
