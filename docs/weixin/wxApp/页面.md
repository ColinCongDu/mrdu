# 页面

## 路由

- `navigateTo`

- `navigateBack`

- `redirectTo`

- `reLaunch`

## 生命周期

小程序页面有 5 个核心的声明周期，按照下面流程图中的顺序调用。

![page-lifecycle](../assets/images/wxapp-page-lifecycle.svg)


- `onLoad`：页面加载

    |特性       |说明                                  |
    |-----------|--------------------------------------|
    |调用次数    |只调用一次。                          |
    |参数        |`query`—— 当前页面路径中的参数对象     |
    |使用场景    | 1. 获取URL参数。<br> 2. 初始化数据请求 |

- `onShow`：页面显示

    |特性       |说明                                  |
    |-----------|--------------------------------------|
    |调用次数    |每次页面显示都触发（可能多次）          |
    |参数        |无                                    |
    |触发时机    | 页面切入前台 / 从其他页面返回          |

- `onReady`：首次渲染完成

    |特性       |说明                                  |
    |-----------|--------------------------------------|
    |调用次数    |只调用一次                            |
    |触发时机    | 	页面初次渲染完成后                    |
    |注意        |对界面进行设置的 API 建议在此之后调用    |


- `onHide`：页面隐藏

    |特性       |说明                                  |
    |-----------|--------------------------------------|
    |调用次数    |只调用一次                            |
    |触发时机    |`navigateTo` 跳走 / Tab 切换 / 小程序切后台    |
    |注意        |不要在这里执行耗时操作，否则影响页面切换流畅度   |


- `onUnload`：页面卸载

     |特性       |说明                                  |
    |---------|--------------------------------------|
    |触发场景 |`redirectTo` / `navigateBack`（且页面被销毁）/ `reLaunch` |
    |注意     |同样避免耗时操作   |


下面是声明周期的使用示例：

```js
Page({
    onLoad(query) {
        // ✅ 只会调用一次
        
        // 典型用途：
        // - 获取跳转传参、
        // - 初始化数据请求
        console.log(query.id)        // 获取路径参数
        console.log(query.type)      // 其他参数
        this.fetchData(query.id)
    },
    onShow() {
        // ✅ 每次页面出现都会触发！
        // 典型用途：
        // - 刷新页面数据（从其他页面返回时）
        // - 恢复暂停的操作（如定时器、音频播放）
        // - 更新用户状态
        this.refreshData()
    },
    onReady() {
        // ✅ 只会调用一次！
        // ✅ 此时页面已渲染完成，可以操作界面
      
      
        // 典型用途：
        // - 设置导航栏标题/颜色（需要等渲染完）
        // - 操作 DOM 节点（如 createSelectorQuery）
        // - 初始化需要界面的组件
        wx.setNavigationBarTitle({ title: '我的页面' })

        const query = wx.createSelectorQuery()
        query.select('#myElement').boundingClientRect(rect => {
        console.log('元素位置:', rect)
        }).exec()
    },
    onHide() {
        // 页面被隐藏/切到后台

        // 典型用途：
        // - 暂停定时器
        // - 暂停音频/视频播放
        // - 保存当前状态（草稿等）
        if (this.timer) {
            clearInterval(this.timer)
        }
        this.saveDraft()
    },
    onUnload() {
        // 页面被销毁（不会回来了）
    
        // 典型用途：
        // - 清理定时器
        // - 取消未完成的网络请求
        // - 解绑事件监听
        // - 断开 WebSocket 连接
        this.cleanup()
    },
    fetchData() {
        // 数据请求...
    },
    refreshData() {
        // 数据刷新  
    },
    cleanup() {
        // 资源清理...
    }
    saveDraft() {
        // 保存草稿...
    }
})
```
