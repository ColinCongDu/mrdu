# 常见错误解答

## 使用css module时找不到模块

你可能会在TypeScript项目中像下面这样使用css module。
```ts
import styles from './index.module.css'
```
然后你就会得到下面这样一个奇怪的错误。
```
Cannot find module './index.module.css' or its corresponding type declarations.
```
这是因为TypeScript默认只支持.ts和.tsx文件。解决方法如下
- 添加类型声明文件。这里的`*.css`也可以修改为`*.module.css`、`*.module.less`、`*.module.scss`等。
```ts
declare module '*.css' {
  const content: { [className: string]: string };
  export default content;
}
```
- 把类型声明文件包含在`tsconfig.json`的`include`中。
```json
{ 
  "include": ["src/**/*"]
}
```
